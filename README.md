# Lashed by Prudyy – Luxury Lash Extensions Website

A modern, responsive landing page for **Lashed by Prudyy**, a luxury lash extension brand.  
Clients can explore services, view lash galleries, check prices, read testimonials, and book appointments online.

The booking system sends appointment requests directly to the business owner's email using **EmailJS**, eliminating the need for a backend server.

---

## 🌟 Features

- Responsive modern design
- Service showcase section
- Lash gallery
- Before & after showcase
- Price list for lash services
- Client testimonials slider
- Appointment booking form
- WhatsApp booking option
- Google Maps location embed
- Email notifications for booking requests

---

## 📧 Booking System

The booking form allows clients to request appointments directly from the website.

**Form fields:**

- Name
- Email
- Phone
- Preferred Date
- Message

**Behavior:**

1. Client fills the form and clicks **Request Booking**
2. Booking request is sent using **EmailJS**
3. The business owner receives the request via email
4. The button updates to show a success message
